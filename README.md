# Book Brazil

**Book Brazil** é um sistema acadêmico de gerenciamento de acervo literário desenvolvido para a disciplina de programação. A aplicação consome uma API REST em tempo real para realizar operações de catálogo dinâmico.

## Objetivo

O projeto visa aplicar conceitos de desenvolvimento front-end, focando na integração entre interface e servidor externo através de requisições assíncrona utilizando JavaScript.

## Design & UI

O projeto adota uma estética **minimalista** e funcional:

**Estética clean:** Uso predominante da cor branca para reduzir a carga cognitiva e focar nos dados.

**Foco no conteúdo:** Layout simplificado para destacar títulos e autores dos livros.

**Interface Direta:** Prioridade na usabilidade das funções de gerenciamento.

## Funcionalidades (CRUD)

O sistema permite o gerenciamento completo dos registros:

**Create (Criar):** Adição de novos livros ao acervo.

**Read (Ler):** Listagem dinâmica dos livros cadastrados na API.

**Update (Atualizar):** Edição de informações de títulos já existentes.

**Delete (Excluir):** Remoção de registros do catálogo.

## Como Executar o Projeto

1. Clone este repositório para sua máquina local.

2. Certifique-se de que os arquivos (index.html) e (logica.js) (ou o seu arquivo de script) estejam na mesma pasta.

3. Abra o arquivo (index.html) em seu navegador (recomendado o uso de extensão **Live Server** no VS code).

4. É necessária uma conexão ativa com a internet para carregar os dados da API.
